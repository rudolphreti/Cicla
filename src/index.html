How does it work?

In the "Key" field, type any text that contains all the characters that appear in your message. You can shuffle these characters by clicking on the "Shuffle" button. The key must also contain characters that repeat in the message. Next, in the "Message" field, enter the message you want to encrypt. Click on "Encrypt this message." The "Message encrypted" field contains the encrypted message, which is unbreakable unless someone knows the key. Notice that the numbers that make up the encrypted message correspond to the character position numbers in the key. To decrypt the message, click the "Decrypt this message" button. The "Message decrypted" field should contain the decrypted message.
